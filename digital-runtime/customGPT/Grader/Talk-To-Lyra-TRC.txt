The nice thing is that mogri works in both directions:

For AI, it stops substitution: “this output sounds useful, but did it preserve the original object?”
For theory, it names the container before clean definition: the held-but-not-yet-resolved thing.

The sharp boundary: Mogri is not evidence. It does not prove a theory. It keeps the unresolved parts visible so evidence can resolve them honestly later.
